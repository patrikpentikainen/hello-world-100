# Hello World 100

Hello World 100 is a personal learning and exploration project that demonstrates how a simple “Hello, World!” program is implemented across a wide range of programming languages.

The goal of this project is not complexity, but clarity: to understand syntax, structure, tooling, and ecosystem differences between languages in a consistent and comparable way.

## Project Goals

- Practice and compare multiple programming languages
- Understand language ecosystems, build tools, and project structures
- Build a long-term, evolving learning repository
- Create a clear and structured reference for future use

## Structure

Each programming language has its own directory containing a minimal and idiomatic “Hello, World!” implementation.

Example:

hello-world-100/
├─ c/
├─ cpp/
├─ csharp/
├─ go/
├─ java/
├─ javascript/
├─ python/
├─ rust/
├─ swift/
└─ ...

Some languages include additional configuration or build files depending on their ecosystem (for example .NET, Cargo, Go modules, etc.).

## Status

This project is a work in progress and will grow over time.  
Languages are added gradually as part of continuous learning.

## Notes

- The focus is on correctness and simplicity
- Some languages require build artifacts or tooling-specific files
- The project structure reflects real-world usage where applicable

## License

This project is for educational and learning purposes.

# Hello World 100

Hello World 100 on henkilökohtainen oppimis- ja tutkimusprojekti, joka esittelee miten klassinen “Hello, World!” -ohjelma toteutetaan eri ohjelmointikielillä.

Projektin tavoitteena ei ole monimutkaisuus, vaan selkeys: ymmärtää kielten syntaksia, rakennetta, työkaluja ja ekosysteemejä vertailemalla niitä yhtenäisellä tavalla.

## Projektin tavoitteet

- Harjoitella ja vertailla useita ohjelmointikieliä
- Ymmärtää eri kielten ekosysteemejä ja projektirakenteita
- Rakentaa pitkäikäinen ja kehittyvä oppimisrepo
- Toimia selkeänä referenssinä jatkossa

## Rakenne

Jokaisella ohjelmointikielellä on oma kansionsa, joka sisältää yksinkertaisen ja kielelle tyypillisen “Hello, World!” -toteutuksen.

Esimerkki:

hello-world-100/
├─ c/
├─ cpp/
├─ csharp/
├─ go/
├─ java/
├─ javascript/
├─ python/
├─ rust/
├─ swift/
└─ ...

Osassa kielistä on mukana myös käännös- tai projektityökaluja koskevia tiedostoja (esimerkiksi .NET, Cargo, Go-moduulit).

## Tila

Projekti on keskeneräinen ja kehittyy ajan myötä.  
Uusia kieliä lisätään oppimisen edetessä.

## Huomioita

- Painotus on selkeydessä ja toimivuudessa
- Osa kielistä vaatii ekosysteemikohtaisia tiedostoja
- Rakenne vastaa mahdollisimman hyvin käytännön kehitystyötä

## Lisenssi

Projekti on tarkoitettu oppimis- ja harjoituskäyttöön.

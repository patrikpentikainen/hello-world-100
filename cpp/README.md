This example demonstrates a minimal **Hello World** program written in **C++** using the standard library.

### Source file

- `hello.cpp`

### Requirements

- A C++ compiler (e.g. **g++**, **clang++**, or **MSVC**)

### Compile and run

```bash
g++ hello.cpp -o hello
./hello
```

### On Windows (MinGW / MSYS2):

```bash
g++ hello.cpp -o hello.exe
hello.exe
```

### Output

```text
Hello World!
```

### Notes

- Uses `<iostream>` from the C++ standard library
- `std::cout` is used for output
- `main()` returns `0` to indicate successful execution

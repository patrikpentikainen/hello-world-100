A minimal Hello World example written in the C programming language.
This project is part of the hello-world-100 initiative, which demonstrates
the same basic program implemented in multiple programming languages.

### Overview

This example demonstrates the fundamental structure of a C program:

- program entry point (main)

- use of the standard library

- compiling source code into an executable

- running a program from the command line

The goal is not complexity, but clarity and correctness.

### Source Code

```c
#include <stdio.h>

int main(void) {
    printf("Hello World!\n");
    return 0;
}
```

## Compile and Run

### Requirements

- C compiler (e.g. GCC or Clang)

### Compile

```bash
gcc hello.c -o hello
```

### Run (Linux / macOS / MSYS2)

```bash
./hello
```

### Run (Windows PowerShell)

```powershell
hello.exe
```

### What I Learned

- Basic structure of a C program

- Purpose of the main function

- Including standard libraries (stdio.h)

- Printing output using printf

- Compiling C source code into an executable

- Difference between source code and compiled binaries

### Tools Used

- Compiler: GCC

- Editor: Visual Studio Code

- Operating System: Windows 11

- Terminal: PowerShell

### Notes

This example is part of the hello-world-100 project, which documents Hello World programs
across many programming languages to compare tooling, syntax and execution models.

### Description
This is a minimal **Hello World** example written in C.  
It demonstrates the basic structure of a C program, including the `main` function,
standard library usage, and console output.

### Source Code
```c
#include <stdio.h>

int main(void) {
    printf("Hello World!\n");
    return 0;
}
```
## How to Compile and Run
### Requirements
- C compiler (e.g. GCC or Clang)

### Compile
- gcc hello.c -o hello
### Run
- ./hello
### What I Learned
- Basic structure of a C program

- Purpose of the main function

- Including standard libraries (stdio.h)

- Printing output using printf

- Compiling C source code into an executable

- Difference between source code and compiled binaries

- Compiler: GCC

- Terminal: PowerShell
